# 🌍 Travel & Relocation Support Portal

A premium, modern, and highly responsive web application built with **Next.js 16**, **React 19**, **TypeScript**, **Tailwind CSS**, and **Framer Motion**. This platform offers comprehensive solutions for global relocation, shipping, freight, and visa services.

---

## ✨ Features & Services

The portal provides an end-to-end user experience for several key relocation and travel services:

*   **✈️ Air Freight** – Fast and reliable international air cargo shipping services.
*   **🚢 Sea Freight** – Cost-effective ocean freight solutions for large shipments and containers.
*   **🚗 Car Shipping** – Secure auto transport and shipping across borders.
*   **🔌 Electronics Shipping** – Specialized shipping for delicate electronic equipment and appliances.
*   **🛃 Customs Clearance** – Professional brokerage services to navigate customs regulations and documentation.
*   **🏠 Relocation Support** – Personalized guidance, packing assistance, and settling-in support.
*   **🛍️ Shop For Me** – Assisted buying service to purchase items internationally and ship them to your door.
*   **🎓 Study Visa** – Expert advisory and documentation assistance for students studying abroad.
*   **💼 Work Permit** – Comprehensive visa processing and legal guidance for professionals.

## 🚀 Getting Started

### 1. Prerequisites
Ensure you have [Node.js](https://nodejs.org/) installed (v18.x or later is recommended).

### 2. Install Dependencies
Run the following command in the project root to install the required npm packages:
```bash
npm install
```

### 3. Run the Development Server
Start the local development server:
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) in your browser to view the application.

### 4. Build for Production
To create an optimized production build:
```bash
npm run build
```
To run the production build locally:
```bash
npm run start
```
